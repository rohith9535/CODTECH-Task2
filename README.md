# Real-Time Chat Application

This project is a **Real-Time Chat Application** developed as part of my internship work at **CODTECH IT SOLUTIONS**. The application allows multiple users to exchange messages instantly through a simple and responsive chat interface powered by **React**, **Vite**, and **WebSocket** technology.

The project demonstrates the implementation of real-time communication between users, live message broadcasting, connection status updates, local message storage, and a clean user-friendly interface. It is designed to show how modern frontend tools can be integrated with a lightweight Node.js WebSocket server to build interactive web applications.

## Features

- Real-time messaging using WebSocket
- React-based responsive user interface
- Username prompt for each user session
- Live connection status indicator
- Auto-scroll to latest messages
- Local storage support for preserving chat history
- Simple Node.js backend for message broadcasting

## Tech Stack

- React
- Vite
- Node.js
- WebSocket (`ws`)
- CSS

## Project Structure

- `src/` - React frontend components and WebSocket client logic
- `server.js` - WebSocket server for handling and broadcasting messages
- `package.json` - Project scripts and dependencies

## How to Run

1. Open the project folder:

```bash
cd chat-app
```

2. Install dependencies:

```bash
npm install
```

3. Start the frontend:

```bash
npm run dev
```

4. Start the WebSocket server in a separate terminal:

```bash
npm run server
```
